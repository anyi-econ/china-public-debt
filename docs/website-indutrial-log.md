## 1. 栏目定义与筛选标准

收录目标：面向**企业**的专门政策/服务入口，强调惠企、政策兑现、营商环境改革、企业服务。

### 1.1 可接受类型（按优先级从高到低）
1. **专门企业服务/政策兑现平台**（独立站或二级域名）
   - 如北京"政策兑现专区" `zhengce.beijing.gov.cn`
   - 黄浦"企业发展服务平台" `zzcx.shhuangpu.gov.cn`
2. **惠企政策 / 涉企政策专栏**
3. **优化营商环境专题**
4. **免申即享 / 政策直达 / 一键兑现** 等主题页
5. **企业服务 / 政务服务中"法人服务"入口**（不首选，但可作兜底）

### 1.2 不收录
- 单篇惠企政策通知（例如某年某项奖励兑现通知）
- 临时活动（如"百日惠企行动"）的专题页
- 早已停更的旧版平台
- 与企业服务无关的普通政策文件列表（那类放在"地区政策"）

## 4. 难点类型（本模块最难）

1. **入口深藏**：惠企政策平台几乎从不放在门户一级导航，通常需要进入：
   - 首页"专题专栏" → "优化营商环境" → 某子栏目
   - 首页"政务服务" → "法人服务" → "涉企政策"
   - 首页"营商环境" → "政策兑现"
   - 二级域名（`qyfw.*`, `zhengce.*`, `zzcx.*` 等）
2. **跳转统一平台**：部分地区没有独立惠企平台，跳转到省级"粤政图谱""惠企通"等，需要判断是否本地过滤。
3. **品牌命名多样**：蓉易享、惠企通、津惠企、亲清在线、渝快办、政策兑现、免申即享、一键申报……没有标准名词。
4. **实名/登录门槛**：不少平台需企业账号登录才能浏览，本导航只需指向主页，不下钻需登录页。
5. **平台迁移**：惠企政策政策平台改版较频繁，旧 URL 经常失效。

## 5. 优先级判定

1. 独立运营的**惠企政策平台 / 政策兑现平台**（二级域名、可搜索、可申报）。
2. 政府门户"优化营商环境"专题首页。
3. "涉企政策""惠企政策"专栏列表。
4. （下策）政务服务网"法人办事"入口。
5. 若以上均无，标记为待复核，**不要用单篇文件凑数**。

## 7. 查找方法

1. 从门户首页"专题专栏""营商环境""政务服务法人办事"三条路径手动排查。
2. 搜索关键词组合（站外 Google）：
   - `{地名} 惠企政策 兑现` / `{地名} 企业服务平台` / `{地名} 政策兑现`
3. 尝试二级域名猜测：`zhengce.`, `zzcx.`, `qyfw.`, `hqzc.`, `qinqing.` + 主域。
4. 如静态页拿不到，用 Playwright 渲染后抓取导航。
5. 批量方法**不推荐**用于此类——每个地区的惠企平台结构差异过大。

## 8. 最终采用原则

- 同一地区存在多入口时，优先"功能最丰富、数据最完整、政府背书属性最强"的平台。
- 企业服务平台 > 专题 > 政策公告。
- URL 含 session/token 的保留仅在 prompt 明确要求时（如黄浦 `zzcx` 示例）。

## 10. 后续扩展建议

1. **省会+副省级**：先集中完成 15 个副省级/计划单列 + 27 个省会的惠企政策平台，预计人均耗时较高，但价值最大。
2. **地级市**：以省为单位批处理，每省抽 30% 人工复核。
3. **区县级**：暂不推进大规模，优先覆盖直辖市辖区、一线城市辖区、经济大县。
4. 脚本草稿：`scripts/website_management/pro-business-probe.mjs`，扫描常见二级域名 + 营商环境专题 URL 模式。
5. 每季度回访已采用链接，检测 404 / 改版，更新本日志。
