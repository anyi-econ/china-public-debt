## 人工复核建议

### 优先级 1：修复错误官网（5 条）
以下官网 URL 已确认指向错误地区，需手动搜索正确的政府门户后再查找预决算链接：
- **盐津县**（云南昭通）— yanjin.gov.cn 指向河南延津
- **彝良县**（云南昭通）— ylx.gov.cn 指向大理云龙
- **富宁县**（云南文山）— funing.gov.cn 指向江苏阜宁
- **连南瑶族自治县**（广东清远）— ln.gov.cn 指向辽宁省
- **北湖区**（湖南郴州）— wap.beihu.gov.cn 为 WAP 移动版

### 优先级 2：安徽 CMS 路径验证（~19 条）
安徽省所有 CMS 均使用 `/zwgk/czyjssgjfxxgk/index.html` 路径（已在滁州 3 县验证）；合肥、蚌埠等市的县区大概率也使用此路径，但因 WAF 拦截无法自动验证。
**建议**：使用浏览器手动访问 `https://www.XXX.gov.cn/zwgk/czyjssgjfxxgk/index.html`

### 优先级 3：云南/湖南大批量县（~70 条）
这些县的官网网络不稳定或使用 JS 渲染，需从当地网络或使用浏览器手动访问。
可尝试的通用路径：
- 云南：`/zwgk/` → 搜索"预决算"；或 `/xljylx/` 路径（丽江模式）
- 湖南：`/zwgk/zdlyxxgk/` → 搜索"财政预决算"

### 优先级 4：特殊行政区划（~7 条）
金门县、三沙市（西沙区/南沙区）、措勤县可标记为 N/A。
新疆兵团新设市需持续关注。
